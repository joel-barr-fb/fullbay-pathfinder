Agent roster — the droids running the loop
C-3PO · Protocol Agent
Assigned to each truck at fleet acquisition. Initiates and manages the handshake. The loop starts the moment C-3PO detects a fault.
Always on
R2-D2 · Routing Agent
GPS/ArcGIS-integrated. Finds the right shop for this fault type -- not just the nearest one. Cross-references bay availability, tech certification, parts inventory.
Pre-arrival
BB-8 · Parts Agent
Checks Unit Pack first. Pack installed: verified parts list, high confidence. No pack: catalog-based with confidence scoring. Auto-orders top match, flags alternatives.
Service loop
L3-37 · Repair Monitoring
The most visible agent. Tracks book time vs. actual in real time. If the job finishes early, the fleet manager knows before anyone calls.
Most human-facing
K-2SO · Verification
Independent post-repair asset self-check. Both K-2SO and shop QA must pass before the release chain fires. Blunt, accurate, not negotiable.
Departure gate
Lobot · Payment
Front-loads the entire payment cycle to fault detection. Checks payment history before the shop agrees to take the unit. Settlement is instantaneous at departure.
Front-loaded
IG-11 · Enforcement
Physical lock at arrival. Requires dual authorization from K-2SO and Lobot to release. Does not negotiate.
Dual-key release
Chopper · Fleet Disposition
Notifies the fleet manager at fault detection, not at arrival. Keeps the view current at every state change. Grumpy, reliable, never misses an update.
Always on
K-3PO · Bay Scheduling
Confirms bay assignment during pre-negotiation before the truck moves. Morning chaos eliminated before anyone arrives.
Pre-arrival
MC80A · Orchestrator
Watches all agent states simultaneously. Detects conflicts. Intervenes only when the system is about to break.
